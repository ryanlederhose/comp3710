# CNN for LFW Dataset
A CNN classifier was constructed for the LFW dataset with two convolutional layers of 3x3 with 32 filters each and dense layers for classification. The CNN uses the following hyperparameters:
* Learning rate = 0.0001
* Batch size = 64
* Number of Epochs = 40
* Optimiser = Adam
* Criterion = Cross Entropy Loss
